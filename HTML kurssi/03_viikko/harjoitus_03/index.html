About the Course


The programming training is designed for adults who want to work as software developers. The training is highly practical compared to similar university-level studies. The studies focus entirely on acquiring the key skills essential for a software developer. The training covers only the most relevant parts of the software developer qualification — those that are essential for the actual work of a software developer.


  • HTML – Page structure and content
  • CSS – Design and layout styling
  • JavaScript – Dynamic and interactive elements
  • React – Component-based JavaScript library
  • PHP – Back-end programming
